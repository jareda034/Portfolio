Project Overview

Website Summary: EldenGuides is a responsive website created for Elden Ring players looking for guides, resources, and helpful tools. The website provides organised navigation to different areas of the game while maintaining a consistent visual design inspired by Elden Ring's atmosphere and aesthetic.

Core Goal: The goal of the project was to design and develop an easy-to-navigate gaming website that demonstrates responsive web design principles, user-focused layouts, and consistent styling while providing useful information for players.

Development Context: This project was developed to improve my front-end web development skills using HTML and CSS. It focused on responsive layouts, website navigation, visual consistency, accessibility, and creating a user-friendly browsing experience.

What I Learned: During development I significantly improved my understanding of

  • Responsive web design
  • HTML semantic structure
  • CSS layouts
  • Navigation design
  • Dropdown menus
  • User experience design
  • Visual consistency
  • Responsive navigation
  • Website accessibility
  • Page layout planning

Biggest Challenge: One of the biggest challenges was designing a navigation system that worked well across both desktop and mobile devices. Creating responsive dropdown menus while keeping navigation intuitive and visually clean required careful planning of the page layout and user experience.

Technical Notes

EldenGuides was developed using HTML and CSS with a strong focus on responsive web design and user experience. The website features semantic HTML structure, responsive navigation, dropdown menus, and consistent page layouts that adapt across desktop and mobile devices. A game-inspired colour palette, imagery, and embedded media were used to create an immersive browsing experience while maintaining readability and accessibility. Throughout the project I strengthened my understanding of front-end development, responsive layouts, CSS styling, navigation design, and creating user-focused websites.
