Project Overview

Game Summary: Marsh's Run is a fast-paced 2D platformer focused on precision movement and time management. Players must navigate hazards while collecting coins that extend the countdown timer and reaching checkpoints before progressing to the finish.

Core Goal: The project was designed around creating responsive and satisfying movement mechanics, with wall jumping and wall sliding encouraging creative platforming and rewarding player skill.

Development Context: This project was developed over three weeks using Unity and C#. It focused on building polished platforming mechanics while introducing risk-versus-reward gameplay through a countdown timer and checkpoint system.

What I Learned: During development I gained experience implementing controller support, checkpoint systems, event-driven programming, collision detection, 2D physics, UI development, and balancing movement mechanics for responsive gameplay.

Biggest Challenge: Balancing the timer system was the largest challenge. Players lose 30 seconds when they die, so careful tuning of coin values and checkpoint placement was required to create tension without making the game feel unfair..

Gameplay Features

Movement Systems

The player movement system focuses on responsive controls and precision platforming. Alongside standard running and jumping, players can wall slide and wall jump to access difficult areas and overcome environmental challenges..

Implemented Features

  • Responsive player movement.
  • Ground and air movement.
  • Wall slide mechanic.
  • Wall jump mechanic.
  • Controller support.
  • Physics-based movement using Unity's Rigidbody2D.

Level Flow

Levels were designed to gradually introduce mechanics before combining them into more challenging platforming sections. Checkpoints reduce frustration while encouraging players to continue improving their movement..

Design Highlights

  • Gradual introduction of movement mechanics.
  • Checkpoint placement to reduce repetition.
  • Risk-versus-reward coin placement.
  • Hazards that encourage mastery of movement.

Timer and Penalties

The core gameplay revolves around completing the level before the timer expires. Collecting coins rewards additional time, while dying removes valuable seconds, creating constant pressure throughout the level.

Gameplay Systems

  • Countdown timer.
  • 30-second death penalty.
  • Three collectible coin types.
  • Bonus 30 seconds awarded every 40 collected coins.
  • Risk-versus-reward gameplay loop.
